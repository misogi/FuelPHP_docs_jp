Query Building

メモ: クエリビルダは、SQLの方言に対応する際に書かれました。

Query Builder (クエリビルダ)は、データベースの問い合わせにあたり、SQL文の記述の抽象化を行います。 クエリビルダはいくつかクラスが分かれていて、クエリを組み立てる際にそれぞれ役割があります。

新しくクエリビルダオブジェクトを作る際、ほとんど DB クラスが メソッドを提供しているように扱うことができ、正しいクエリビルダを得られるように導いてくれます。 常識的なやり方で、コードの可読性を保ってくれます。

クエリビルダは以下のクラスからなります。:

  • Query_Builder (abstract)
  • Query_Builder_Where (abstract)
  • Query_Builder_Select
  • Query_Builder_Insert
  • Query_Builder_Update
  • Query_Builder_Delete
  • Query_Builder_Join
